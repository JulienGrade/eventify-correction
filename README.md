# Eventify - Correction complète

Ce dépôt contient :

- Backend Spring Boot sécurisé avec JWT
- Frontend React (Eventify)

## Lancer le backend

cd eventify
./mvnw spring-boot:run

## Lancer le frontend

cd eventify-frontend
npm install
npm run dev
